# ClinicFlow

## Descrição
O ClinicFlow é um sistema simples em linha de comando para cadastro de pacientes e agendamento de consultas em clínicas pequenas e consultórios.

## Problema real
Pequenas clínicas e profissionais autônomos frequentemente organizam pacientes e consultas de forma manual ou desorganizada, o que pode gerar erros, retrabalho e dificuldade no acompanhamento dos atendimentos.

## Proposta da solução
A aplicação permite registrar pacientes, criar agendamentos, listar consultas e atualizar o status dos atendimentos, ajudando na organização do fluxo básico de uma clínica.

## Público-alvo
- clínicas pequenas
- consultórios
- profissionais da saúde autônomos
- recepcionistas e atendentes

## Funcionalidades
- cadastro de pacientes
- listagem de pacientes
- busca de paciente por ID ou nome
- agendamento de consultas
- listagem de agendamentos
- atualização de status
- cancelamento de agendamento

## Tecnologias utilizadas
- Python
- JSON
- Pytest
- Ruff
- GitHub Actions
